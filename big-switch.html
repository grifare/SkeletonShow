The Big Switch or D-day

DESCRIPTION

Make it fun.

INSTRUCTIONS

  1. Build a simple HTML page that uses a few elements that could be used to build a toggle switch.
  2. Style the switch using CSS so that it looks convincing.
  3. Incorporate some sort of CSS transform, transition effect or a simple CSS animation to some part of the switch.
  4. Employ one or more jQuery effects to some aspect of the user interface.
  5. Have fun.

According to rumours, CSS now is said to be capable of handling even onclick events.

USERS' MANUAL - In total, there are 3 buttons a user can click.

  • The GREEN button on the left, is intended to showcase how a CSS-driven event handler works. No JavaScript at all!
  • The currently turned-off button in the middle that reads "OFF", is a 2-image set provided by the instructor and its functioning is purely jQuery-driven.
  • The analogue switch underneath, is the actual deal. It is NOT an image in the first place: it is purely CSS. Once clicked, CSS3 and jQuery hybridly handle a hell lot of events, together. BEWARE!
  • Have fun.
